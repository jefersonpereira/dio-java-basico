O sistema recebe dois parâmetros via terminal (dois números inteiros).
Com a diferença do segundo parâmetro pelo primeiro obteremos a quantidade de interações (for) para realizar a impressão no console.

Se o primeiro parâmetro for MAIOR que o segundo parâmetro, haverá uma exceção customizada chamada de ParametrosInvalidosException com a segunda mensagem: "O segundo parâmetro deve ser maior que o primeiro" ou você pode digitar a palavra "sair" para encerrar.

Caso você queira continuar é só obedecer as mensagens de comando ou digitar "sair" após a impressão no console
